✨ New Pill Navigation Features

Your navigation has been upgraded with a modern, animated pill-style design inspired by ReactBits. This implementation uses vanilla JavaScript and GSAP for smooth, professional animations.

  • Circular hover effect that grows from the bottom of each pill
  • Text slides up on hover with a smooth animation
  • Logo rotates 360° when you hover over it
  • Mobile-responsive hamburger menu with slide-down animation
  • Fully accessible with proper ARIA labels and keyboard navigation
  • GSAP-powered animations with customizable easing
  • Automatic active state detection based on current page
  • Smooth entrance animation on page load

How to Test

Desktop: Hover over each navigation pill to see the circular expansion effect and text animation.

Mobile: Resize your browser to <768px or use responsive mode to see the hamburger menu.

Logo: Hover over "Psychoanalytically Speaking" to see the rotation animation.

Customization

The pill navigation colors are configured to match your existing design system:

  • Base Background: #FAFAF8 (Off-white)
  • Pill Background: #4A3828 (Brown-dark)
  • Pill Text: #FAFAF8 (Off-white)
  • Hover Text: #FAFAF8 (Off-white)
