## [1.0.0] - 2026-01-11

### Added
- Initial release of Psychoanalytically Speaking website
- Modern pill navigation with GSAP animations
- WebGL silk background effect using Three.js
- Split-text character/word reveal animations
- Scroll-based reveal animations
- Mobile-responsive hamburger menu
- 11 HTML pages (Home, About, Approach, Consultation, Contact, Blogs, Readings, Lectures, E-books, Reading Groups, Special Requests)
- Complete design system with CSS custom properties
- Comprehensive documentation (README, Architecture, Design System, Deployment)
- Optimized images for psychology-themed decorative graphics
- Accessibility features (ARIA labels, keyboard navigation, reduced motion support)
- SEO-friendly meta tags and semantic HTML

### Components
- **PillNavigation**: ReactBits-inspired animated navigation
- **SilkBackground**: WebGL shader-based animated backgrounds
- **SplitText**: Intersection Observer-powered text animations
- **ScrollReveal**: Progressive content disclosure

### Styles
- `base.css`: CSS reset and normalization
- `design-system.css`: Design tokens and variables
- `components.css`: Reusable UI components
- `pages.css`: Page-specific styles  
- `animations.css`: Animation and effects
- `decorative.css`: Decorative graphics layout
- `pill-nav.css`: Pill navigation styles

### Scripts
- `main.js`: Core interactions and utilities
- `pill-nav.js`: Pill navigation component
- `silk-background.js`: WebGL shader background
- `split-text.js`: Text animation component

### Documentation
- `README.md`: Project overview and quick start
- `docs/ARCHITECTURE.md`: Technical architecture
- `docs/DESIGN_SYSTEM.md`: Design guidelines
- `docs/DEPLOYMENT.md`: Deployment guide
- `LICENSE`: MIT License
- `CHANGELOG.md`: This file

## How to Update This Changelog

When making changes:

1. **Added** for new features
2. **Changed** for changes in existing functionality
3. **Deprecated** for soon-to-be removed features
4. **Removed** for now removed features
5. **Fixed** for any bug fixes
6. **Security** in case of vulnerabilities

Example:
```markdown
## [1.1.0] - 2026-02-01

### Added
- Blog search functionality
- RSS feed for blog posts

### Changed
- Updated pill navigation hover timing to 0.2s

### Fixed
- Mobile menu z-index issue on iOS Safari
```
